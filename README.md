# Calculator – Project Information

# This project is a basic calculator web application created using HTML, CSS, and JavaScript.
It allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division.

# HTML is used to create the structure of the calculator.

# CSS is used to design and style the calculator layout.

# JavaScript is used to handle button clicks and perform calculations.

# The calculator has a display screen to show numbers and results, number buttons (0–9), operator buttons, a clear button, and an equal button.
This project helps beginners understand how frontend technologies work together.
